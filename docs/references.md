# References

This page lists a starter set of external references that KoT aligns with or builds on. Add sources and links as KoT documentation evolves.

* [RDF](https://www.w3.org/RDF/) on w3.org: Resource Description Framework overview
* [JSON-LD](https://json-ld.org/) on json-ld.org: JSON-based serialization for linked data
* [PROV](https://www.w3.org/TR/prov-overview/) on w3.org: Provenance data model and documents
* [Web of Things](https://www.w3.org/WoT/) on w3.org: W3C Web of Things standards and deliverables
* [Schema.org](https://schema.org/) on schema.org: Shared vocabulary for structured data
* [Wikidata](https://www.wikidata.org/) on wikidata.org: Collaborative knowledge base
* [Wikimedia Foundation](https://wikimediafoundation.org/) on wikimediafoundation.org: Wikimedia governance and operations
* [OriginTrail](https://origintrail.io/) on origintrail.io: Decentralized knowledge graph infrastructure
